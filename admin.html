Data Collection Status

⚠️ **IMPORTANT: Data is not stored on the server.** Due to the limitation of a backend-less GitHub Pages site, the user data is collected manually.

How to Collect Data Log:

  1. Instruct the person testing the compatibility to click the **"Download Data Log"** link after getting their result.
  2. The user's browser will download a file named `compatibility_log.csv`.
  3. The log file will contain the names, result, and a timestamp for that single test.
  4. To see **all** data, you need to collect and merge the `compatibility_log.csv` files sent to you by all the users.

**Example Log Content:**
`Timestamp,Name_A,Name_B,Compatibility_Result`
`2025-10-06T13:00:00Z,"John Doe","Jane Smith",Lovers`
